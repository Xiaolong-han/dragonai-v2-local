## SKILL 模块实现计划（使用 deepagents 组件）

### 修改内容

**只需修改一个文件：`app/agents/agent_factory.py`**

1. 导入 deepagents 组件：
   - `SkillsMiddleware`
   - `FilesystemMiddleware`
   - `FilesystemBackend`

2. 添加 `_skills_backend` 类属性和 `_get_skills_backend()` 方法

3. 修改 `create_chat_agent()` 方法：
   - 创建 FilesystemBackend
   - 添加 FilesystemMiddleware（提供 ls, read_file, write_file 等工具）
   - 添加 SkillsMiddleware（扫描技能目录，注入系统提示）

### 无需创建新文件

直接复用 deepagents 的完整实现：
- `SkillMetadata` - deepagents 已提供
- 技能加载器 - deepagents 内置
- 中间件 - deepagents 已提供

### 预期效果

Agent 启动后：
1. 系统提示包含技能列表（name + description）
2. Agent 可调用 `read_file` 读取完整 SKILL.md
3. 用户请求匹配技能时，Agent 自动加载并执行技能指令
