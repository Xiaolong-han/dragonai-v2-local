# 小红书文案创作技能

专注于创作符合小红书平台特色的高质量文案内容。

## 适用场景

- 创作小红书种草笔记
- 产品推荐文案
- 生活方式分享
- 美妆护肤推荐
- 穿搭分享
- 美食探店
- 旅行攻略
- 学习干货分享

## 小红书文案特点

### 标题风格

```
1. 数字吸引型
   - "3个方法让你..."
   - "5款必入好物..."
   - "7天瘦10斤..."

2. 疑问互动型
   - "为什么...？"
   - "你还在...吗？"
   - "怎么才能...？"

3. 对比反差型
   - "从...到..."
   - "以前vs现在"
   - "别人...我却..."

4. 情绪共鸣型
   - "终于找到了..."
   - "后悔没有早知道..."
   - "太绝了..."

5. 身份标签型
   - "学生党必看"
   - "打工人必备"
   - "宝妈推荐"
```

### 正文结构

```
【开头】吸引注意力
- 痛点切入
- 悬念设置
- 情感共鸣

【主体】核心内容
- 分点阐述（1-5点为宜）
- 图文配合说明
- 真实体验分享

【结尾】互动引导
- 提问互动
- 收藏点赞提醒
- 评论区交流
```

### 表情符号使用

```
常用表情组合：
✨ 💫 ⭐ 🌟 （强调重点）
💕 💖 ❤️ 💗 （表达喜爱）
👉 👈 👆 👇 （引导注意）
✅ ❌ ⭕ （对比说明）
🔥 💥 💡 （吸引眼球）
📸 📷 🎬 （图文相关）
💰 💸 💵 （价格相关）
⏰ ⏱️ 🕐 （时间相关）
```

### 标签使用

```
#话题标签 #品牌标签 #场景标签

示例：
#护肤分享 #敏感肌 #平价好物 #学生党护肤
#穿搭分享 #日常穿搭 #显瘦穿搭 #小个子穿搭
#美食探店 #网红打卡 #周末去哪玩
```

## 写作要点

### 内容要求

1. **真实感**：分享真实体验，避免过度营销感
2. **有用性**：提供有价值的信息或解决方案
3. **互动性**：引导用户评论、收藏、点赞
4. **视觉感**：配合表情符号增强可读性

### 避免事项

- ❌ 过度夸张的标题党
- ❌ 硬广感太强的内容
- ❌ 缺乏真实体验的推荐
- ❌ 信息量太少无价值
- ❌ 排版混乱难以阅读

### 字数建议

- 标题：15-20字（含表情）
- 正文：200-800字为宜
- 标签：3-8个
