## "技能模块" → "工具模块" 全面重构方案

### 一、后端文件重命名 (4个文件)

| 原文件 | 新文件 |
|--------|--------|
| `app/api/v1/skills.py` | `app/api/v1/tools.py` |
| `app/schemas/skills.py` | `app/schemas/tools.py` |

### 二、后端代码修改

#### 1. `app/api/v1/tools.py` (原skills.py)
- 路由前缀: `/skills` → `/tools`
- 标签: `["技能"]` → `["工具"]`
- 函数名: `get_all_skills` → `get_all_tools`, `get_skill_detail` → `get_tool_detail`
- 参数名: `skill_name` → `tool_name`
- 注释: 所有"技能" → "工具"

#### 2. `app/schemas/tools.py` (原skills.py)
- 类名: `SkillResponse` → `ToolResponse`, `SkillDetailResponse` → `ToolDetailResponse`
- 字段描述: "技能名称" → "工具名称" 等

#### 3. `app/schemas/models.py`
- 类名: `SkillModelResponse` → `ToolModelResponse`
- 字段名: `skill_type` → `tool_type`
- 显示名: "编程技能" → "编程工具", "翻译技能" → "翻译工具"

#### 4. `app/api/v1/models.py`
- 函数名: `get_skill_models` → `get_tool_models`
- 路由: `/skills` → `/tools`
- 变量名: `skill_models` → `tool_models`

#### 5. `app/api/v1/__init__.py`
- 导入: `skills` → `tools`

#### 6. `app/llm/model_factory.py`
- 类名: `AsyncSkillModel` → `AsyncToolModel`
- 注释: "异步技能模型" → "异步工具模型"

#### 7. `app/llm/__init__.py`
- 导出: `AsyncSkillModel` → `AsyncToolModel`

#### 8. `app/main.py`
- 导入: `skills` → `tools`
- 路由注册: `skills.router` → `tools.router`

### 三、前端文件重命名 (2个文件)

| 原文件 | 新文件 |
|--------|--------|
| `frontend/src/stores/skill.ts` | `frontend/src/stores/tool.ts` |
| `frontend/src/components/SkillList.vue` | `frontend/src/components/ToolList.vue` |

### 四、前端代码修改

#### 1. `frontend/src/stores/tool.ts`
- Store名: `useSkillStore` → `useToolStore`
- API路径: `/api/v1/skills/` → `/api/v1/tools/`

#### 2. `frontend/src/components/ToolList.vue`
- 接口名: `Skill` → `Tool`
- 变量名: `skills` → `tools`, `skillRoutes` → `toolRoutes`
- 函数名: `fetchSkills` → `fetchTools`, `getSkillIcon` → `getToolIcon`
- CSS类: `.skill-*` → `.tool-*`
- 文本: "AI 技能" → "AI 工具"

#### 3. `frontend/src/components/ChatInput.vue`
- 接口名: `Skill` → `Tool`
- 变量名: `skills` → `tools`, `activeSkill` → `activeTool`
- 函数名: `selectSkill` → `selectTool`, `clearSkill` → `clearTool`
- CSS类: `.skill-*` → `.tool-*`
- 文本: "选择技能" → "选择工具"

#### 4. `frontend/src/stores/chat.ts`
- 接口名: `SkillOptions` → `ToolOptions`
- 函数名: `sendMessageWithSkill` → `sendMessageWithTool`
- 内部函数: `_sendTranslationSkill` → `_sendTranslationTool` 等
- API路径: `/api/v1/skills/` → `/api/v1/tools/`

#### 5. `frontend/src/views/` (多个文件)
- `Chat.vue`: 更新函数调用
- `Translation.vue`: API路径修改
- `Coding.vue`: API路径修改
- `ImageGeneration.vue`: Store引用和CSS类名修改
- `ImageEditing.vue`: API路径修改
